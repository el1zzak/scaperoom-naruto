# scaperoom-naruto
# Naruto Escape Room VR

## 📌 Descripció

Aquest projecte és un joc tipus *escape room en 3D* desenvolupat amb **A-Frame**, on el jugador controla en Naruto i ha de superar diferents sales, trobar objectes i resoldre reptes per avançar en la història.
